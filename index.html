Connect to Firebase (Free)

Firebase syncs data across all phones in real time. Follow these steps — takes about 3 minutes:

1
Go to console.firebase.google.com and sign in with your Google account
2
Click "Create a project" → name it petrov-tracker → disable Google Analytics → click Create
3
In the left menu click "Realtime Database" → "Create database" → choose any location → select "Start in test mode" → Enable
4
Click the ⚙️ gear icon → "Project settings" → scroll down → click "</>" (web app) → register app → copy the firebaseConfig object values below
